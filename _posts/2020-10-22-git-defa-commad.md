```
### git 사용자 정보 설정
git config user.name "사용자 이름"
git config user.email 이메일주소

```

```
### 깃 저장소 초기화 하고 저상소 생성
git init
```

```
### 수정된 새로생성된 파일 Staging Area 올림
git add *
```

```
### Staging area 에 있는 파일을 커밋
git commit -m "message"
```
